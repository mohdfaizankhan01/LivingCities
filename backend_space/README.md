# LivingCities API

FastAPI backend for the [LivingCities](https://livingcities-app.netlify.app) Life-Centric Urbanism Index.

Serves the scorecard plus live Cognee-memory demos: `/ask`, `/feedback`, `/forget`.

Set `COGNEE_SERVICE_URL` and `COGNEE_API_KEY` as Space secrets.
